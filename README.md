## üìå How To Install?

-  Clone This Repo

```
$  git clone
```

-  Go To Folder Repo

```
$  cd fazzniture-backend
```

-  Setup .env

```
DB_USER
DB_HOST
DB_DATABASE
DB_PASSWORD
DB_PORT
DATABASE_URL
JWT_SECRET
JWT_ISSUER
CLOUD_NAME
CLOUD_API
CLOUD_SECRET
SERVICE
USER
PASS
BASE_URL
```

-  Install Module

```
$  npm install
```

-  To Start The Project

```
$  npm run startDev
```

## üñá Endpoint Details

| Endpoint         |     Method     | Info             |
| ---------------- | :------------: | :--------------- |
| /auth            |     `POST`     | Auth             |
| /users/:id       | `GET` `PATCH`  | Users            |
| /sizes           |     `GET`      | Size             |
| /brands          |     `GET`      | Brands           |
| /categories      |     `GET`      | Categories       |
| /colors          |     `GET`      | Colors           |
